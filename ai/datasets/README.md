# AI 데이터셋 디렉터리

## 디렉터리 기준

- `exports/`: 관리자 API에서 내려받은 원천 export 파일을 둔다. Git 추적 제외.
- `processed/`: 학습용으로 전처리한 파일을 둔다. Git 추적 제외.

## 입력 파일 형식

v0.8.7 기준 `train_baseline.py`는 아래 형식을 지원한다.

- JSON: `{ "rows": [...] }` 또는 배열 `[...]`
- JSONL: 한 줄에 JSON 객체 하나
- CSV: 첫 줄 header, 이후 row

민감 정보는 export 단계에서 마스킹되어야 하며, 학습 파일에 이메일, 전화번호, 주소, 토큰, 비밀번호 원문을 저장하지 않는다.

## 포트폴리오 샘플 데이터

`samples/`에는 실제 운영 데이터가 없는 환경에서 학습/평가 파이프라인을 재현하기 위한 합성 데이터가 들어 있다. 이 데이터는 개인정보를 포함하지 않으며 Git에 포함한다.
