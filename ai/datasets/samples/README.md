# 포트폴리오 AI 샘플 데이터

이 디렉터리의 데이터는 실제 운영 데이터가 아니라 포트폴리오 데모용 합성 데이터다.

## 포함 데이터

- `demo_demand_train.json`: 상품 수요 예측 학습용 합성 데이터
- `demo_demand_eval.json`: 상품 수요 예측 평가용 합성 데이터
- `demo_review_train.json`: 리뷰 감성 점수 학습용 합성 데이터
- `demo_review_eval.json`: 리뷰 감성 점수 평가용 합성 데이터

## 사용 목적

- 실제 고객/주문/리뷰 데이터 없이도 AI 학습 파이프라인을 재현한다.
- `.pt` 체크포인트 생성 흐름을 포트폴리오에서 설명할 수 있게 한다.
- 운영 데이터가 생기면 동일한 스크립트와 config 구조로 확장할 수 있음을 보여준다.

## 보안 기준

- 이름, 이메일, 전화번호, 주소, 주문번호, 토큰, 비밀번호 같은 개인정보를 포함하지 않는다.
- 실제 리뷰 원문을 포함하지 않고 숫자형 feature로만 구성한다.
- 대용량 export 데이터와 모델 산출물은 Git에 커밋하지 않는다.
