# v0.8 AI 데이터셋 및 학습 기반 계획

## 목표

v0.8은 상품, 주문, 재고, 배송, 리뷰, 문의, 회계/정산 데이터를 기반으로 AI 학습용 데이터셋과 `.pt` 모델 학습 구조를 구축하는 단계다.

v0.7에서 정리한 회계/정산 데이터는 매출 예측, 환불/반품 비용 분석, 이상 주문 탐지, 정산 리스크 탐지의 핵심 피처 후보로 이관한다.

## 데이터셋 후보

- 상품: 상품코드, 브랜드, 카테고리, 가격, 태그, 판매 상태, 전시 상태
- 주문: 주문일, 주문 상태, 결제 상태, 총액, 할인, 상품 수량
- 결제/환불: 결제 금액, 환불 상태, 환불 금액, 환불 시점
- 반품: 반품 사유, 승인 여부, 반품 배송비, 부담 주체
- 배송: 택배사, 배송 방법, 송장 발급 시점, 배송 상태, 배송 완료 시점
- 재고: SKU, 안전재고, 재고 이동, 입고, 출고, 재고 실사
- 리뷰/문의: 평점, 리뷰 내용, 숨김 여부, 문의 유형, 답변 여부
- 회계/정산: 매출 거래, 환불 거래, 배송비 수익/비용, 정산 배치, 정합성 이슈

## 보안/운영 원칙

- 학습 데이터는 운영 DB 원본을 직접 사용하지 않고 스냅샷 또는 export 데이터를 사용한다.
- 개인정보는 학습 전에 마스킹 또는 제거한다.
- 모델 산출물은 버전, 학습 데이터 기준일, 주요 지표와 함께 기록한다.
- `.pt` 파일은 Git 추적 대상에서 제외하고, 저장소에는 학습 코드와 문서만 관리한다.
- 실제 추천/예측 자동화는 v0.9 운영 화면 연동 단계에서 진행한다.

## v0.7에서 이관된 후보

- 회계 거래 누락 후보를 이상 데이터 탐지 라벨 후보로 사용한다.
- 환불/반품/배송비 데이터를 손익 리스크 분석 피처로 사용한다.
- 정산 배치와 마감 데이터를 기간별 매출/비용 예측 검증 데이터로 사용한다.
- 권한/감사 로그는 운영자 행동 분석과 보안 이벤트 탐지 후보로 보관한다.
