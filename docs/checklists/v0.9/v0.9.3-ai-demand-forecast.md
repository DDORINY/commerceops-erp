# v0.9.3 AI 수요 예측 체크리스트

## 목적

상품별 재고, 안전재고, 태그/검색 키워드를 기반으로 관리자용 수요 예측 후보와 재고 소진 위험을 표시한다.

이번 버전은 자동 발주/생산 지시가 아니라 포트폴리오 데모용 수요 예측 운영 화면이다.

## API 엔드포인트

| Method | Path | 권한 | 설명 |
| --- | --- | --- | --- |
| `GET` | `/api/admin/ai/forecasts/demand?limit=10` | `AI_FORECAST_READ` | 상품별 수요 예측 후보와 재고 소진 위험을 조회한다. |

## 예측 기준

- `displayStatus=VISIBLE`
- `salesStatus=ON_SALE` 또는 `SOLD_OUT`
- 현재 재고
- 안전재고
- 태그 수
- 검색 키워드 수
