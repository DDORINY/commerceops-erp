# v0.9.5 AI 이상 주문 탐지 체크리스트

## 목적

주문 금액, 할인율, 주문/결제 상태 불일치 후보를 기반으로 관리자용 이상 주문 후보를 표시한다.

이번 버전은 자동 주문 차단이 아니라 운영자가 직접 확인할 주문을 찾는 포트폴리오 데모 지표다.

## 탐지 기준

- 고액 주문
- 높은 할인율
- 주문 취소 상태와 결제 완료 상태 불일치

## 검증 결과

- backend test: 통과 (`.\gradlew.bat test --no-daemon`)
- frontend lint: 통과 (`npm.cmd run lint`)
- frontend build: 통과 (`NEXT_DIST_DIR=.next-build-check-v095 npm.cmd run build`)
- `git diff --check`: 통과
- 참고: 일반 build가 Windows `spawn EPERM`으로 실패해 권한 상승으로 재실행했다. Next가 추가한 `tsconfig.json` 임시 include는 제거했고 `.next-build-check-v095` 산출물도 삭제했다.
