## 남은 이슈와 후속 범위

- 실제 서비스 운영용 AI 모델 성능 검증은 제외한다.
- 운영 데이터 수집/보관과 개인정보 원문 기반 학습은 제외한다.
- 실시간 추론 서버, MLOps 플랫폼, 외부 LLM 자동 업무 처리는 후속 고도화 범위로 유지한다.
- AI 결과는 발주, 주문 차단, 정산 확정 같은 자동 실행에 사용하지 않고 관리자 검토용 참고 지표로 유지한다.
