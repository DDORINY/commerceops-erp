# v0.9.6 AI 리스크 알림 체크리스트

## 목적

재고 부족과 정산 확인 필요 후보를 관리자용 AI 리스크 알림 화면에 표시한다.

이번 버전은 자동 입고/정산 마감이 아니라 운영자가 먼저 확인할 리스크를 찾는 포트폴리오 데모 지표다.

## API 엔드포인트

| Method | Path | 권한 | 설명 |
| --- | --- | --- | --- |
| `GET` | `/api/admin/ai/risks/inventory?limit=10` | `AI_RISK_ALERT_READ` | 재고 부족 리스크 후보를 조회한다. |
| `GET` | `/api/admin/ai/risks/settlement?limit=10` | `AI_RISK_ALERT_READ` | 정산 확인 필요 리스크 후보를 조회한다. |

## 검증 결과

- backend test: 통과 (`.\gradlew.bat test --no-daemon`)
- frontend lint: 통과 (`npm.cmd run lint`)
- frontend build: 통과 (`NEXT_DIST_DIR=.next-build-check-v096 npm.cmd run build`)
- `git diff --check`: 통과
- 참고: 일반 build가 Windows `spawn EPERM`으로 실패해 권한 상승으로 재실행했다. Next가 추가한 `tsconfig.json` 임시 include는 제거했고 `.next-build-check-v096` 산출물도 삭제했다.
