# v0.9.4 AI 리뷰 분석 체크리스트

## 목적

리뷰 평점과 마스킹된 본문을 기반으로 관리자용 리뷰 감성 분석 후보를 표시한다.

이번 버전은 자동 숨김/제재가 아니라 운영자가 확인할 리뷰를 찾는 포트폴리오 데모 지표다.

- backend test: 통과 (`.\gradlew.bat test --no-daemon`)
