# v0.7.4 택배비 매입/배송비 정산 체크리스트

작업 버전: `v0.7.4-shipping-cost-settlement`

## 목표

택배사/배송 방법의 기본 배송비를 기준으로 배송 건별 택배비 매입 비용을 회계 거래로 반영한다.

## 정책

- 택배비 비용은 `shipping_methods.default_fee`를 기준으로 한다.
- `Shipment.carrier` 문자열과 `Carrier.code/name`을 매칭해 배송 방법을 찾는다.
- 매칭되는 배송 방법이 없으면 첫 번째 활성 배송 방법을 fallback 후보로 사용한다.
- 배송 방법 기본비가 0 이하이면 회계 거래를 생성하지 않는다.
- 고객 청구 배송비 매출은 주문 테이블에 배송비 컬럼이 없어 이번 버전에서는 생성하지 않는다.

## 제외 범위

- 실제 택배사 정산 파일 수집.
- 주문별 고객 청구 배송비 매출 자동 계산.
- 배송 방법별 복잡한 운임 정책.
- 지역/중량/부피별 배송비 계산.
- 정산 배치 마감 연결.
