# v0.5.4 Barcode Stock Processing Checklist

## Purpose

Barcode based SKU lookup from v0.5.3 is extended into simple stock processing. Admin users can scan or enter a barcode, view SKU/product/warehouse stock, and process simple inbound/outbound stock changes. Existing `Product.stockQuantity`, `WarehouseStock`, and `InventoryLog` are reused; a separate `inventory_movements` table is deferred.
