# v0.5.5 Stock Count Adjustment Checklist

## Purpose

Add a minimal stock count workflow. Admin users can create a warehouse stock count session, add SKU items, save counted quantities, calculate differences, complete the session, and apply differences to existing `Product.stockQuantity`, `WarehouseStock`, and `InventoryLog`.
