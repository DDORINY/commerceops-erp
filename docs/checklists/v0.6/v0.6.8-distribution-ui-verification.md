## 확인 결과

- 출고 관리 화면은 목록/상세/생성/수정/피킹/취소와 바코드 검수 입력을 제공한다.
- 택배사/배송 방법 화면은 목록/검색/생성/수정/활성 상태 변경을 제공한다.
- 배송 관리 화면은 송장번호 수동 입력/자동 생성/수정, 송장 라벨 미리보기/출력 이력, 배송 추적 이벤트, 상태 변경을 제공한다.
- 반품 관리 화면은 승인/거절과 수거 송장/상태/배송비 정보를 관리한다.
- 권한 기준은 `OUTBOUND_READ`, `OUTBOUND_MANAGE`, `CARRIER_MANAGE`, `SHIPMENT_READ`, `SHIPMENT_MANAGE`, `SHIPPING_LABEL_PRINT`, `RETURN_SHIPPING_MANAGE` 흐름을 따른다.

## 남은 UI/UX 이슈

- 실제 브라우저 수동 클릭 검증은 별도 실행 환경에서 보완한다.
- 실제 택배사 API, 프린터 SDK, 산업용 스캐너 SDK 연동은 후속 범위다.
- 출고 완료 기준 재고 ledger 통합과 회계 매출 확정은 v0.7 이후 범위로 이관한다.
