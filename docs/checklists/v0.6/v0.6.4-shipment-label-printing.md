## 구현 요약

- 송장 라벨은 `shipment_labels` 테이블에 배송 건, 송장번호 snapshot, 택배사, 라벨 형식, 출력 횟수, 마지막 출력 시각, 생성자를 저장한다.
- 기본 라벨 형식은 `SHIPMENT_100X150`이다.
- 라벨 생성은 송장번호와 택배사가 있는 배송 건에서만 가능하다.
- 화면은 실제 프린터 SDK/PDF 출력이 아니라 HTML 미리보기와 출력 이력 기록만 제공한다.

## API

- `GET /api/admin/shipments/{shipmentId}/labels`: 배송 건의 송장 라벨 이력 조회, `SHIPMENT_READ`
- `POST /api/admin/shipments/{shipmentId}/labels`: 송장 라벨 생성, `SHIPPING_LABEL_PRINT`
- `POST /api/admin/shipments/labels/{labelId}/print`: 출력 이력 기록, `SHIPPING_LABEL_PRINT`
