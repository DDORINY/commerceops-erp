## 구현 결정

- 마이그레이션 도구는 Flyway를 사용한다.
- 초기 DDL은 `backend/src/main/resources/db/migration/V1__initial_schema.sql`에 둔다.
- 운영 기준 DB는 MySQL 8.0으로 둔다.
- 기존 H2 기반 테스트 안정성을 유지하기 위해 `application-test.yml`에서는 Flyway를 비활성화한다.
- 기존 개발 DB가 Hibernate `ddl-auto=update`로 생성되었을 가능성을 고려해 `baseline-on-migrate=true`, `baseline-version=0` 설정을 둔다.
- `application-local.yml`은 커밋 제외 대상이므로 수정하지 않았다.
