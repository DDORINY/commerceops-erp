## API

- `GET /api/admin/ai/datasets`
- `GET /api/admin/ai/datasets/{key}/export?limit=100`

지원 key:

- `PRODUCTS`
- `ORDERS`
- `REVIEWS`
- `ACCOUNTING_TRANSACTIONS`
