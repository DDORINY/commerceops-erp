## 기존 구조 확인 요약

- `Shipment`는 현재 주문 1:1 구조로 송장번호, 택배사, 배송 상태, 출고일, 배송 완료일을 관리한다.
- `Order`와 `OrderItem`은 주문/주문 품목의 기준이며, v0.6 출고 지시는 주문 이후 단계로 분리한다.
- `Sku`는 상품과 연결된 재고/바코드 단위이며, v0.6 출고 품목과 바코드 검수의 기준으로 사용한다.
- `WarehouseStock`, `WarehouseLocationStock`, `InventoryLog`는 출고 완료 시 재고 차감 정책과 연결할 후보 구조다.
- `PermissionChecker`, `PermissionCodes`, `AuditLogService`는 v0.6 관리자 API 권한과 감사 로그 기록에 재사용한다.
