## 정책

- 공개 상단 네비는 `active=true` and `visibleInNav=true` 카테고리만 표시한다.
- 관리자 tree API는 active/visible 여부와 관계없이 전체 카테고리를 반환한다.
- 삭제 API는 v0.3.3에서 만들지 않고 `active=false` 비활성화를 우선한다.
- 상품 등록/수정 UI는 `GET /api/categories` 기준 active 카테고리만 선택지로 보여준다. 기존 상품이 비활성 카테고리를 참조하는 경우 DB 참조는 유지한다.
- parent 변경 시 자기 자신 parent 지정과 순환 참조를 금지한다.
