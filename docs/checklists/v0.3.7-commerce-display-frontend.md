## 남은 이슈

- 상품 상세 SEO metadata는 현재 client component 구조상 직접 반영하지 않았다. 서버 컴포넌트 분리 또는 metadata route 설계는 후속 작업으로 이관한다.
- 메인 상품 섹션은 현재 공개 상품 목록 API의 정렬/필터를 기반으로 단순 구성한다. 별도 전시 섹션/기획전 CMS는 후속 범위다.
- 실제 수동 브라우저 확인은 별도 실행하지 못했으며 lint/build 기준으로 검증했다.
