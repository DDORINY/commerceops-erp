## 사이드바 그룹 구조

- 대시보드: 대시보드, 운영 분석, 알림
- 쇼핑몰 관리: 카테고리 관리, 상단 네비 관리, 배너 관리, 사업자 설정, 약관 설정
- 상품 관리: 상품 관리, 상품 등록, 상품 상태 관리
- 주문/배송/CS 관리: 주문 관리, 배송 관리, 반품 관리, 환불 관리, 문의 관리, 리뷰 관리
- 재고/창고 관리: 재고 관리, 창고 관리, 입고 관리, 재고 이동
- 매출/회계 관리: 매출 통계, 회계 관리, 결제 내역, 환불 내역, 정산 관리
- 마케팅 관리: 쿠폰 관리, 이벤트 관리, 프로모션 관리
- 인사/권한 관리: 직원 관리, 부서 관리, 직급 관리, 권한 그룹 관리, 역할/권한 설정
- 시스템 설정: 사업자 설정, 약관 설정, 개인정보처리방침 설정, 배송/반품 정책 설정, 관리자 작업 이력, 감사 로그, 환경 설정

## 권한별 메뉴 노출

- `SUPER_ADMIN`: 전체 그룹과 전체 메뉴 노출
- `ADMIN`: 운영/상품/주문/재고/매출/마케팅/시스템 설정 대부분 노출
- `MANAGER`: 조회 중심 메뉴 노출
- `USER` 또는 비로그인: 기존 `AdminLayout`에서 관리자 접근 차단

## v0.3.5.2 사이드바 아코디언 UX 보정

- `Record<string, boolean>` 기반 다중 open 상태를 제거하고 `openGroupLabel` 단일 상태로 변경했다.
- 현재 `pathname`과 query string이 일치하는 메뉴의 그룹만 기본으로 열린다.
- query string이 있는 메뉴는 `pathname + searchParams`가 모두 일치해야 active 처리된다.
- `/admin/sales`는 매출/회계 관리의 매출 통계에만 남겼다.
- 사업자/약관/개인정보/배송 정책 설정은 시스템 설정에만 남겼다.
- 같은 `/admin/settings/audit-logs`를 쓰던 관리자 작업 이력/감사 로그는 관리자 작업 이력 하나로 통합했다.
