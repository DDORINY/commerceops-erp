# v0.4.2 Staff Management API/UI Checklist

## 목적

관리자가 직원 계정과 인사 프로필을 조회/관리할 수 있도록 직원 관리 API와 관리자 직원 관리 화면을 구현한다.

## 브랜치

- 작업 브랜치: `v0.4.2-staff-management-api-ui`

## 정책

- [x] `GET /api/admin/staff/**`는 `ADMIN`, `SUPER_ADMIN` 접근 허용
- [x] `POST/PATCH /api/admin/staff/**`는 `SUPER_ADMIN` 전용
- [x] `MANAGER`는 직원 관리 접근 불가
- [x] 생성 가능한 직원 role은 `MANAGER`, `ADMIN`, `SUPER_ADMIN`으로 제한
- [x] 일반 `USER` 계정 생성은 직원 관리 API에서 제외
- [x] 이메일 변경과 비밀번호 변경은 v0.4.2에서 제외
- [x] 자기 자신의 `SUPER_ADMIN` 권한 제거와 마지막 활성 `SUPER_ADMIN` 약화 작업 방어
- [x] `staff_profiles.active`는 직원 프로필 활성 상태이며 로그인 차단은 기존 `users.status`가 담당
