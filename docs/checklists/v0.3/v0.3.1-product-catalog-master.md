## 상품 마스터 필드 설계

- `price`: 판매가. 기존 주문/장바구니 흐름과 호환을 위해 유지한다.
- `originalPrice`: 정상가.
- `discountPrice`: 할인 금액. `price`보다 클 수 없다.
- `purchasePrice`: 매입가. 관리자 API와 관리자 화면에만 노출한다.
- `marginRate`: 저장하지 않고 관리자 응답에서 `(price - purchasePrice) / price * 100`으로 계산한다.
- `searchKeywords`, `tags`, `seoKeywords`: v0.3.1에서는 TEXT 구분자 문자열로 저장한다.
