## 남은 이슈

- 운영 메모 수정/삭제는 v0.3.6 범위에서 제외하고 후속 운영 고도화로 이관한다.
- 상품 상태 변경 이력은 판매 상태/전시 상태가 실제 변경된 경우에만 기록한다. 안전재고만 변경한 경우 이력은 남기지 않는다.
- MANAGER는 SecurityConfig 기준으로 관리자 상품 GET API만 접근 가능하며, 대량 상태 변경/운영 메모 작성은 ADMIN/SUPER_ADMIN으로 제한된다.
- 전체 관리자 기능 감사 로그 확장은 v0.4 이후 권한/감사 로그 고도화 범위로 유지한다.
