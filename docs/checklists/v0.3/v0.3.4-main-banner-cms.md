## 정책

- 공개 배너 API는 `active=true`이고 현재 시간이 `startsAt <= now <= endsAt` 범위에 있는 배너만 반환한다.
- `startsAt` 또는 `endsAt`이 `null`이면 열린 기간으로 처리한다.
- 관리자 API는 비활성/기간 종료 배너까지 조회한다.
- 삭제 API는 실제 삭제가 아니라 `active=false` 비활성화로 처리한다.
- 배너 이미지는 v0.3.4에서 URL 입력 방식으로 최소 구현한다. 배너 전용 업로드는 후속 이슈로 남긴다.
- 관리자 배너 GET은 `MANAGER`, `ADMIN`, `SUPER_ADMIN` 조회 가능, 생성/수정/삭제는 `ADMIN`, `SUPER_ADMIN`만 허용한다.

## 남은 이슈

- 관리자 배너 이미지는 URL 입력 방식이다. 배너 전용 이미지 업로드/리사이징/S3-CDN 연동은 후속 버전으로 이관한다.
- 일부 기존 프론트 파일에는 이전 인코딩 문제로 보이는 깨진 한글 문구가 남아 있다. 이번 작업에서 신규/수정 배너 화면과 사이드바의 주요 문구는 정상 한국어로 정리했다.
- 브라우저 수동 확인은 dev server 실행 환경에서 `/admin/banners`와 `/` 중심으로 추가 확인한다.
