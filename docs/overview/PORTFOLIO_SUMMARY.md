# 포트폴리오 요약

CommerceOps ERP는 쇼핑몰과 ERP 운영 흐름을 결합한 AI 커머스 ERP 포트폴리오 프로젝트다. 단순 CRUD가 아니라 상품 전시, 주문, 결제, 재고, 생산 입고, 바코드, 출고, 송장, 배송 추적, 회계, 권한, 감사 로그, AI 운영 화면까지 이어지는 운영 데이터 흐름을 구현했다.

## 강조할 점

- MVP 이후 각 버전을 체크리스트와 태그로 관리했다.
- 사용자 화면과 관리자 화면의 mock 데이터를 실제 API 중심으로 정리했다.
- 상품/전시/운영관리, 인사/권한관리, 재고/생산/바코드, 유통/배송까지 단계적으로 확장했다.
- 권한은 role 기반에서 permission code 기반으로 세분화했다.
- 주요 관리자 작업은 audit log로 추적한다.
- v0.8/v0.9에서는 AI 데이터셋, baseline `.pt` 모델, 관리자 AI 운영 화면, 포트폴리오 데모 시나리오까지 확장했다.
