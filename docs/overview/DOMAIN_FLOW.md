# 도메인 흐름

## 쇼핑몰 구매 흐름

```text
상품 조회 -> 장바구니 -> 주문 생성 -> 결제 승인 -> 주문 상태 변경 -> 출고/배송
```

## 관리자 상품/전시 흐름

```text
상품 등록 -> 상품 마스터 입력 -> 상세 블록 작성 -> 카테고리/배너 노출 -> 판매/전시 상태 관리
```

## 재고/생산/바코드 흐름

```text
SKU 생성 -> 바코드 발급 -> 생산 주문 -> 생산 완료 입고 -> 창고 재고 증가 -> 바코드 입출고
```

## 출고/배송 흐름

```text
주문 확인 -> 출고 지시 생성 -> 바코드 검수 -> 송장번호 생성/입력 -> 라벨 출력 -> 배송 상태 추적
```

## 반품/환불 흐름

```text
반품 요청 -> 관리자 승인/거절 -> 반품 배송 정보 입력 -> 재고/회계 연동 후보 -> 환불 처리
```

## 권한/감사 로그 흐름

```text
관리자 요청 -> role 1차 확인 -> permission code 확인 -> 작업 수행 -> audit log 기록
```

## AI 후보 흐름

```text
운영 DB -> dataset export -> 전처리 -> 모델 학습 -> model 파일 -> FastAPI 추론 -> 관리자 화면
```
