- v0.6은 출고 지시, 택배사/배송 방법, 송장번호, 송장 라벨, 배송 상태 추적, 반품/교환 배송, 바코드 출고 검수를 중심으로 진행한다.
- v0.6.0에서 v0.6.1 ~ v0.6.9 작업 순서와 DB/API/프론트/권한/감사 로그 후보를 한국어 문서로 확정했다.
- 배송비, 환불 배송비, 택배비 매입/매출, 출고 완료 기준 매출 확정 후보는 v0.7 회계/정산 고도화로 이관한다.
- 실제 택배사 API 완전 연동, 실제 프린터/스캐너 SDK, 자동 운임 계산, 실시간 GPS 배송 추적은 v0.6 제외 범위로 유지한다.
