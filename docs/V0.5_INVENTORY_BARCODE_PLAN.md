# v0.5 재고관리/생산 입고/바코드 자동화 계획 후보

기준 버전: `v0.4.9-hr-permission-release-verification`

문서 상태: v0.4.9 검증 중 정리한 v0.5 이관 후보 문서다. v0.5.0에서 세부 범위와 제외 범위를 확정한다.

## 목표

v0.5는 상품/주문/창고 기반 위에 실무 ERP의 재고 흐름을 축소 구현한다. 생산 입고, 바코드 생성, 바코드 라벨 출력, 재고 실사, 창고 위치 관리, 안전재고 알림을 중심으로 재고 운영 흐름을 고도화한다.

## v0.5 세부 후보

| 버전 | 주제 | 포함 후보 |
| --- | --- | --- |
| v0.5.0 | 재고/생산/바코드 계획 수립 | 범위, DB/API/UI 설계, 제외 범위 확정 |
| v0.5.1 | SKU/바코드 재고 마스터 고도화 | SKU 코드, 바코드 값, 상품-옵션-SKU 연결 |
| v0.5.2 | 생산 입고 모델 | 생산 지시, 생산 완료, 생산 입고, 재고 자동 증가 |
| v0.5.3 | 바코드 생성/라벨 출력 | 바코드 자동 생성, 라벨 프린트용 화면/PDF 후보 |

## 핵심 요구사항 후보

- 상품을 생산하는 경우 생산 입고 가능
- 생산 완료 시 재고 자동 증가
- 바코드 자동 생성
- 바코드 라벨 출력
- 바코드 기반 재고 조회
- 입고/출고/조정/이동 이력 기록
- 안전재고 기준과 자동 알림 후보
- v0.6 유통관리, 송장 자동화와 연결 가능한 구조
